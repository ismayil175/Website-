Introduction

A database is a planned grouping of data that has been organized and is typically kept electronically in a computer system. A database management system typically oversees a database (DBMS). The term "database system," which is frequently abbreviated to "database," refers to the combination of the data, the DBMS, and the applications that are connected to them.

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install MAX-dt

NPM

$ npm install MAX-dt

Yarn

$ yarn install MAX-dt
